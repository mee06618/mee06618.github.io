  - c 형식 지정자
    - %d -> 10진수 정수 (4바이트) -> int



    - %ld -> 10진수 정수 (8바이트) -> long



    - %f -> 실수형 (4바이트) -> float



    - %lf -> 실수형 (8바이트) -> double



    - %c -> 문자 -> char (1바이트)



    - %s -> 문자열 -> char*



    - %o -> 8진수 -> int(4바이트)



    - %x -> 16진수 -> int



    - %p -> 주소값 -> void*



    - %u -> 부호없는 10진수 (4바이트) -> unsigned int



    - %lu -> 부호없는 10진수 (8바이트) -> unsigned long
